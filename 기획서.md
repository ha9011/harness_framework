# 기획서: 영어 패턴 학습기

## 1. 프로젝트 개요

영어 단어와 패턴을 등록하고, AI(Gemini)가 둘을 조합해 난이도별 예문을 생성하며,
간격 반복(Spaced Repetition) 카드 복습으로 기억을 정착시키는 웹 기반 학습 도구.

## 2. 문제 정의

| 문제 | 설명 |
|------|------|
| 단어만 외우면 문장이 안 됨 | 단어를 알아도 패턴 속에서 활용하지 못하면 회화에서 쓸 수 없음 |
| 교재 예문이 한정적 | 교재에 있는 10개 예문만으로는 패턴이 체화되지 않음 |
| 복습이 체계적이지 않음 | 언제 뭘 복습해야 하는지 모르고, 다 까먹고 나서야 다시 봄 |
| 등록이 귀찮음 | 직접 타이핑하면 귀찮아서 안 하게 됨 |

## 3. 솔루션

- **AI 예문 생성**: 내가 등록한 단어 × 패턴을 AI가 조합해서 무한한 예문 생성
- **이미지 추출**: 교재 사진/유튜브 캡쳐만 올리면 AI가 자동으로 텍스트 추출
- **간격 반복 복습**: 까먹기 전에 자동으로 복습 카드 제시
- **양방향 카드**: 영→한(인식) + 한→영(회상) 양방향으로 진짜 실력 향상

## 4. 대상 사용자

영어 패턴 교재로 공부하는 학습자. 교재 사진을 찍어서 등록하고, 유튜브 캡쳐로 단어를 등록.

## 5. 핵심 사용 흐름

```
📸 교재 사진 촬영 / 유튜브 캡쳐
       ↓
🤖 Gemini Vision → 텍스트 추출 (패턴/단어 자동 파싱)
       ↓
✅ 사용자 확인 후 저장
       ↓
🔄 난이도 + 개수 선택 → AI가 단어×패턴 조합 예문 생성
       ↓
📖 예문 학습
       ↓
🃏 간격 반복 카드 복습 (매일 자동 선정)
```

## 6. 핵심 기능 (MVP)

### 6.1 단어 등록
- 직접 입력 (단어 + 뜻)
- JSON 벌크 등록 (여러 단어 한번에)
- 이미지 업로드 → AI 추출 (유튜브 캡쳐 등)
- AI 자동 보강: 품사, 발음, 유의어, 한줄 팁
- 중요 체크(⭐) 토글

### 6.2 패턴 등록
- 직접 입력 (패턴 + 한국어 설명)
- 교재 사진 업로드 → AI 추출 (패턴 + 설명 + 예문 10개 + 해석)

### 6.3 예문 생성
- 난이도(초급/중급/고급) + 개수(10/20/30) 선택
- 단어 선택 우선순위: ⭐중요 > 복습 적은 것 > 랜덤
- 패턴은 랜덤 조합
- 예문마다 감정이입할 수 있는 **상황(situation)** 함께 생성
  - 예문당 5개 상황 생성, 볼 때마다 랜덤 1개 표시
  - 예: "택시 안에서 친구에게 전화하는 상황"
  - 예문 표시 시 "이 상황을 상상하며 읽어보세요" + 상황 설명
- 단어 상세에서 개별 예문 생성도 가능

### 6.4 복습 (간격 반복)
- 하루 복습 개수 설정 (10/20/30)
- 카드 선정: 모름 우선 > 오래 안 본 것 > 복습 적은 것 > 타입별 균등 배분
- 양방향 카드 (인식: 영→한 / 회상: 한→영)
- 기억남/애매/모름 응답 → SM-2 알고리즘으로 간격 자동 조정

### 6.5 학습 기록
- 단어/패턴 등록 시 자동 생성
- 날짜별 학습 이력 (Day N, 총 학습일 기준)

### 6.6 홈 대시보드
- 오늘의 복습 현황 + [복습 시작] 버튼
- 누적 통계 (단어, 패턴, 예문, 복습 연속일)
- 최근 학습 목록

## 7. Phase 1 — 회원가입/로그인 + 데이터 격리

### 7.1 회원가입/로그인
- 회원가입: 이메일(UNIQUE) + 비밀번호(최소 8글자, BCrypt) + 닉네임. 가입 즉시 자동 로그인
- 로그인: 이메일 + 비밀번호 → JWT(HttpOnly Cookie, 24시간)
- 로그아웃: Cookie 삭제
- 미로그인 시 로그인 페이지로 리다이렉트 (홈/학습 화면 접근 불가)
- 홈 화면에 "닉네임님 안녕하세요" 표시

### 7.2 사용자별 데이터 격리
- 모든 학습 데이터(단어, 패턴, 예문, 복습, 학습기록, 설정)가 사용자별 독립
- 다른 사용자의 데이터에 접근 불가

### 7.3 복습 카드 모던 스타일
- FlipCard 앞면: 크림 배경 + 강화 shadow
- FlipCard 뒷면: 라떼 소프트 배경 (#E8D5BE)

## 8. 향후 버전
- 닉네임/비밀번호 변경
- TTS/STT 말하기 기능 (Web Speech API, 비용 0)
- 퀴즈/빈칸 테스트
- 즐겨찾기
- 이미지 묘사 학습 — AI 이미지 생성 → 수준별 묘사 예문. TOEIC Speaking/OPIC 대비

## 9. 디자인 방향
- 모바일 우선 반응형
- 심플하고 깔끔한 UI
- 복습 카드는 플립 애니메이션
