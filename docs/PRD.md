# PRD: 영어 패턴 학습기

## 목표
영어 단어와 패턴을 함께 등록하고, Gemini가 생성한 상황 기반 예문과 간격 반복 복습으로 실제 회화 활용력을 높이는 모바일 우선 웹 학습 도구를 만든다.

## 사용자
- 영어 패턴 교재로 공부하며 단어를 문장 안에서 자연스럽게 쓰고 싶은 학습자
- 교재 사진, 유튜브 캡처, 직접 입력을 섞어 학습 자료를 빠르게 등록하고 싶은 사용자
- 매일 복습할 카드가 자동으로 정리되는 개인용 학습 시스템이 필요한 사용자

## 성공 기준
- 사용자는 회원가입/로그인 후 자신의 단어, 패턴, 예문, 복습 기록만 조회하고 조작할 수 있다.
- 사용자는 단어와 패턴을 직접 입력, 벌크 입력, 이미지 추출 흐름으로 등록할 수 있다.
- 사용자는 난이도와 개수를 선택해 단어×패턴 조합 예문을 생성하고, 예문별 상황 5개를 저장할 수 있다.
- 사용자는 단어/패턴/문장 탭별 복습 카드를 진행하고, 기억남/애매/모름 응답에 따라 다음 복습일이 갱신되는 것을 확인할 수 있다.
- 홈 대시보드는 오늘 남은 복습 수, 누적 통계, 최근 학습 기록, 커피나무 성장 상태를 한 화면에서 보여준다.

## 핵심 기능
1. 회원가입/로그인
   - 이메일, 비밀번호, 닉네임으로 가입한다.
   - 비밀번호는 BCrypt로 저장하고 JWT는 HttpOnly Cookie로 전달한다.
   - 가입 즉시 자동 로그인하고, 미로그인 사용자는 학습 화면에 접근할 수 없다.
   - 모든 학습 데이터는 사용자별로 격리하며, 다른 사용자의 리소스 접근은 403으로 막는다.
2. 단어 등록과 관리
   - 단어와 뜻을 단건 등록한다.
   - JSON 배열로 여러 단어를 벌크 등록하고 중복 단어는 스킵한다.
   - 이미지 업로드로 단어를 추출하며 이미지는 저장하지 않는다.
   - Gemini가 품사, 발음, 유의어, 한줄 팁을 보강한다.
   - 중요 체크, 검색, 품사 필터, 중요 단어 필터, 등록일/단어 정렬을 제공한다.
   - 삭제는 soft delete로 처리하고, 해당 단어의 WORD 복습 카드만 비활성화한다.
3. 패턴 등록과 관리
   - 패턴 템플릿과 한국어 설명을 직접 등록한다.
   - 교재 사진에서 패턴, 설명, 교재 예문, 해석을 추출한다.
   - 교재 예문 순서는 보존한다.
   - 삭제는 soft delete로 처리하고, 해당 패턴의 PATTERN 복습 카드만 비활성화한다.
4. 예문 생성
   - 난이도는 유아, 초등, 중등, 고등을 지원한다.
   - 생성 개수는 10, 20, 30개를 지원한다.
   - 단어 선택 우선순위는 중요 체크, 복습 횟수 낮음, 랜덤 순서다.
   - 패턴은 등록된 패턴 중 랜덤 조합한다.
   - Gemini 응답에는 사용한 wordId, patternId, 영어 예문, 한국어 해석, 상황 5개가 포함되어야 한다.
   - 단어 상세과 패턴 상세에서도 해당 항목 중심 예문을 생성할 수 있다.
5. 복습
   - 단어, 패턴, 문장 탭을 분리하고 각 탭에서 설정값 N개씩 독립 선정한다.
   - 선정 기준은 오늘 복습 대상, HARD 우선, 오래 안 본 것, 복습 횟수 낮음 순이다.
   - WORD/PATTERN은 인식과 회상 양방향 카드, SENTENCE는 인식 카드만 만든다.
   - 복습 응답은 EASY, MEDIUM, HARD이며 SM-2 기반 커스텀 알고리즘으로 interval_days, ease_factor, next_review_date를 갱신한다.
   - 완료 후 같은 카드를 읽기 전용으로 다시 보거나, 이미 복습한 카드를 제외하고 추가 복습을 요청할 수 있다.
6. 학습 기록과 대시보드
   - 단어/패턴 등록 시 해당 날짜의 학습 기록을 자동 생성한다.
   - day_number는 실제 학습한 날짜 수 기준으로 계산한다.
   - 대시보드는 누적 단어/패턴/예문 수, 복습 연속일, 오늘 남은 복습 수, 최근 학습 기록을 제공한다.
7. 설정
   - 하루 복습 개수는 10, 20, 30 중 선택한다.
   - 설정은 DB에 저장되어 로그인한 사용자의 모든 환경에서 동일하게 적용된다.

## MVP 제외 사항
- 닉네임/비밀번호 변경
- TTS/STT 말하기 기능
- 퀴즈/빈칸 테스트
- 즐겨찾기
- 이미지 묘사 학습
- 알림, 데이터 백업, 데이터 내보내기

## 디자인
- 모바일 우선 반응형 UI로 출퇴근길 한 손 사용을 우선한다.
- 디자인 컨셉은 Cozy Cafe & Coffee Tree다.
- 기본 팔레트는 크림 화이트 배경, 오트밀/크라프트 표면, 라떼 브라운 포인트, 세이지 그린 보조색, 모카 브라운 텍스트다.
- 복습 카드는 메뉴판/냅킨 느낌의 플립 카드로 제공한다.
- 예문 상황은 세이지 톤 구름 말풍선으로 표시한다.
- 홈에는 복습 연속일과 연결되는 커피나무 성장 상태를 표시한다.
